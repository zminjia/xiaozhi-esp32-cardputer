# 显示屏问题排查指南

## 可能的问题

1. **引脚配置错误** - 当前配置为推测值
2. **SPI 模式不正确** - 当前使用模式 0，可能需要模式 3
3. **缺少显示屏开启命令** - 可能需要调用 `esp_lcd_panel_disp_on_off`
4. **背光未开启** - 背光引脚可能不正确
5. **复位引脚未配置** - 可能需要硬件复位

## 需要验证的配置项

### 当前配置（需要验证）：
- SCK: GPIO 18
- MOSI: GPIO 19  
- DC: GPIO 15
- CS: GPIO 5
- RESET: NC (未配置)
- BACKLIGHT: GPIO 38

### 常见 ST7789 配置：
- SPI 模式: 通常为模式 0 或模式 3
- 需要调用 `esp_lcd_panel_disp_on_off(panel, true)` 开启显示
- 可能需要硬件复位引脚

## 建议的修复步骤

1. 添加显示屏开启命令
2. 尝试不同的 SPI 模式（0 或 3）
3. 检查是否需要复位引脚
4. 验证背光控制
